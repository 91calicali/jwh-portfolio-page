저는 아이디어를 시각화하여 전달하는 일에 관심이 많습니다. 단순히 눈으로 보는 것 뿐이지만 사용자와 가장 가깝게 소통할 수 있는 수단이라고 생각하기 때문입니다. 그렇기에 좋은 사람들과 좋은 아이디어를 인상 깊게 전달하는 일을 해보고 싶습니다.

  • Javascript, jQuery를 사용해 동적인 웹을 구현할 수 있습니다.

HTML5 CSS3 SCSS Javascript jQuery Vue.js

Java Spring MySQL MyBatis
